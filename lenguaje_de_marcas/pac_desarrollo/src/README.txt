- Nombre del proyecto:
    PAC Desarrollo: Lenguaje de Marcas - Javi Escobar Fernández (Compra entradas grupo musical Estopa)

- Descripción:
    Un sitio web que muestra una pequeña información del grupo Estopa.
    Incluye un formulario interactivo que permite a los usuarios seleccionar detalles como fecha y lugar del concierto, categoría y cantidad de entradas,
    además del método de pago. Tras completar el formulario, los datos ingresados son almacenados y se muestran en la página de confirmación.

- Tecnologías utilizadas:
    HTML, CSS y JavaScript.

- Requisitos:
    Este proyecto requiere un navegador moderno (compatibilidad con HTML5, CSS3 y JavaScript) para una experiencia óptima.

- Navegadores compatibles:
    Probado en Google Chrome, Mozilla Firefox y Microsoft Edge.

- Instrucciones / como ejecutarlo:
    Descargar el proyecto completo y ejecutar el archivo index.html en un navegador web.

- Estructura de carpetas y archivos:
    index.html: Página principal, es el archivo que se debe ejecutar para iniciar el funcionamiento
    confirmacion.html: Página secundaria, es donde se redirecciona una vez se han comprado las entradas correctamente para mostrar los datos de la compra.
    css: directorio que contiene los ficheros de estilo que se aplican a la página web.
    css/style.css: archivo principal para establecer y ajustar el estilo visual de la página web.
    js: directorio que contiene los ficheros de JavaScript que se ejecutan en la página web.
    js/script.js: fichero principal para dar funcionalidad a la página web.
    img: directorio que contiene todas las imágenes utilizadas en la página web.
    img/*.png / *.jpg / *.ico: Todas las imagenes utilizadas en la web.
